# ask-zenon-wtf

A repository containing Zenon Network of Momentum research documentation, designed to power an interactive Q&A interface for exploring protocol design concepts.

## Source Attribution

The research documents in this repository are sourced from [TminusZ/zenon-developer-commons](https://github.com/TminusZ/zenon-developer-commons). All credit for the original research and documentation goes to TminusZ and contributors to that project.

## Purpose

This repository serves as the knowledge base for a user interface that allows users to ask questions about Zenon protocol design research. The documentation covers:

- Verification-first architecture and bounded verification
- Dual-ledger system design (account-chains + momentum chain)
- Bitcoin SPV integration research
- Light client protocols
- Proof-native applications (zApps)
- Node architecture (Pillars, Sentinels, Sentries, Supervisors)

## Structure

- `context/` - All research documentation files
- `CLAUDE.md` - AI assistant guidance for working with this repository
